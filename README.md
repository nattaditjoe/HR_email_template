# Universal HR Email Template Library (Bilingual TH/EN)

A production-ready React + Vite + Tailwind site that provides 22 modern-formal HR email templates across the full recruitment lifecycle.

## Features
- Bilingual templates (Thai + English)
- Search by situation/tags
- Filter by stage and tone
- Live variable rendering and one-click copy
- Lightweight, fast, deploys to Vercel/Netlify in minutes

## Quick Start
```bash
npm install
npm run dev
```
Then open http://localhost:5173

## Build
```bash
npm run build
npm run preview
```

## Deploy (Vercel)
- Push this folder to GitHub
- Import the repo in https://vercel.com → Deploy

## Structure
- `src/templates.js` — all 22 templates (TH/EN)
- `src/App.jsx` — UI & logic
- Tailwind preconfigured
